# Wedding Background Music

Place your wedding music file here.

## Required file:

- **File:** `wedding-song.mp3`
- **Format:** MP3
- **Bitrate:** 128-192 kbps
- **Duration:** 3-5 minutes recommended

## Music suggestions:

### Classical/Instrumental:
- Canon in D - Pachelbel
- Air on the G String - Bach
- Clair de Lune - Debussy
- Wedding March - Mendelssohn

### Modern Instrumental:
- A Thousand Years - Piano Cover
- All of Me - Piano Cover
- Perfect - Ed Sheeran Instrumental
- Marry You - Bruno Mars Instrumental

## Converting to MP3:

If you have a different format (WAV, M4A, etc):

1. **Online converter:**
   - CloudConvert: https://cloudconvert.com
   - Online-Convert: https://www.online-convert.com

2. **VLC Player:**
   - Media > Convert/Save
   - Select your file
   - Choose MP3 format
   - Convert

## Important notes:

- Music will NOT auto-play (browser policy)
- Users must click the ♪ button to play
- Keep file size under 10MB for faster loading
- Test on different browsers/devices
