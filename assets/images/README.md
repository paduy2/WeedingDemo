# Wedding Images

Place your wedding photos here:

## Required files:

### Hero Image
- **File:** `hero.jpg`
- **Size:** 1920x1080px or larger
- **Description:** Main full-width photo displayed at the top

### Gallery Images
- **Files:** `gallery-1.jpg` to `gallery-6.jpg`
- **Size:** 800x1000px (4:5 ratio - portrait)
- **Description:** Photo gallery images

## Image optimization tips:

1. Use online tools:
   - TinyPNG: https://tinypng.com
   - Squoosh: https://squoosh.app

2. Recommended settings:
   - Format: JPG
   - Quality: 80-85%
   - Target file size: < 500KB per image

3. Responsive considerations:
   - Hero image should work well on both desktop and mobile
   - Avoid important content at edges (will be cropped on mobile)

## Example structure:

```
images/
├── hero.jpg           (1920x1080px)
├── gallery-1.jpg      (800x1000px)
├── gallery-2.jpg      (800x1000px)
├── gallery-3.jpg      (800x1000px)
├── gallery-4.jpg      (800x1000px)
├── gallery-5.jpg      (800x1000px)
└── gallery-6.jpg      (800x1000px)
```
